# SmartHire – AI Resume Analyzer

> Where resumes meet intelligence and hiring meets precision.

Built and deployed AI-powered resume analyzer with experience in integrating ML-based text processing.
##  Features

- Upload Resume (PDF/DOC)
-  AI-based Resume Analysis
-  ATS Score Calculation
-  Skill Extraction
-  Role-based Suggestions
-  Recruiter Candidate Ranking
-  Real-time Processing

## 🛠 Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- Multer (file upload)
- AI-based text processing

### Deployment
- Frontend: Vercel
- Backend: Render

---

## ⚙️ How It Works

1. Upload your resume
2. System extracts text
3. AI analyzes skills & experience
4. Generates ATS score
5. Suggests improvements
